# 🏢 Employee Performance Predictor

## 🚀 Overview

This project is an **AI-powered HR Analytics System** that predicts employee performance using Machine Learning and provides interactive insights through a professional dashboard.

---

## 🎯 Problem Statement

Companies struggle to:

* Identify high-performing employees
* Detect low performance early
* Make data-driven HR decisions

This project solves these problems using **data analytics and machine learning**.

---

## 💼 Business Value

* 📊 Helps HR teams make better decisions
* 🚀 Identifies high performers for promotions
* ⚠️ Detects low performers for training
* 📈 Improves workforce productivity

---

## 🧠 Features

* 🔮 Employee Performance Prediction (ML Model)
* 📊 Interactive HR Dashboard
* 📈 Data Insights & Visualizations
* 🧮 Confidence Score (Prediction Probability)
* 🧭 Multi-page Web App (Dashboard, Prediction, Insights, About)
* 🎨 Clean UI with professional layout

---

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn (Random Forest)
* Streamlit
* Plotly, Matplotlib, Seaborn

---

## 🏗 Project Architecture

```
Employee Data → Data Processing → ML Model → Prediction → Dashboard → HR Insights
```

## 📊 Dashboard Preview

## ⚙️ How to Run

### 1. Clone Repository

```
git clone https://github.com/your-username/employee-performance-predictor.git
cd employee-performance-predictor
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Run Model (Generate Data + Model)

```
python main.py
```

### 4. Run Dashboard

```
streamlit run app.py
```
